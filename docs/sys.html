Dictonary Lookup Results

Below are four examples of return results for the two nouns, Hund and Geschäft, and the two verbs, befragen and kommmen.

For verbs that have one or more prefix versions, their results also are also returned. Therefore lookup('kommen', 'de', 'en') returns an array of 59 results that includes *ankommen, aufkommen, etc, along with kommen, which is the 24th result element.

The Systran JSON matche object has these properties:

  "model_name": "string",
  "source": {
    "inflection": "string",
    "info": "string",
    "lemma": "string",

    "pos": "string",
    "term": "string"
  },
  "target": {
    "context": "string",
    "domain": "string",
    "entry_id": "string",
    "expressions": [
      {
        "source": "string",

    ],

      "string"
    ],
